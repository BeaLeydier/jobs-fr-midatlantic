# Jobs FR
Offres d'emploi dans les entreprises françaises de DC · Maryland · Virginie · Pennsylvanie · Virginie-Occidentale · Delaware 

## Détails 
Liste mise à jour automatiquement chaque lundi avec les offres d'emploi publiées sur les sites des entreprises françaises de la région (liste d'entreprises selon le [rapport du SER 2025](https://media.franceintheus.org/wp-content/uploads/2025/05/2025_Economic_Report.pdf)). Pour les entreprises dont les sites ne sont pas compatibles avec notre méthode de scraping des données, un lien vers le site carrière est inclus plutôt que les offres individuelles. Ainsi, le nombre d'offres trouvées sous-estime le total d'offres publiées, et nous vous invitons à visiter les sites carrière de chaque entreprise qui vous intéresse et dont les offres d'emploi ne sont pas reflétées individuellement.

## A propos
Code de scraping des offres d'emploi développé en Python avec Code Claude. Pour toute question ou suggestion sur le produit, écrire à Béatrice Leydier (beatrice@bealeydier.com).
